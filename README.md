## 📋 Описание программы

**Weather Diary** — это графическое приложение для ведения дневника погоды. Программа позволяет записывать ежедневные наблюдения за погодой, фильтровать записи по дате и температуре, а также сохранять данные в формате JSON.

### ✨ Основные возможности

- ➕ **Добавление записей** с указанием:
  - Даты (формат ГГГГ-ММ-ДД)
  - Температуры (числовое значение)
  - Описания погоды
  - Наличия осадков (да/нет)

- ✅ **Валидация вводимых данных**:
  - Проверка формата даты
  - Проверка, что температура — число
  - Проверка на пустое описание

- 🔍 **Фильтрация записей**:
  - По конкретной дате
  - По минимальной температуре
  - Комбинированная фильтрация

- 💾 **Работа с JSON**:
  - Автоматическое сохранение при добавлении
  - Загрузка данных при запуске
  - Обработка повреждённых файлов

- 🎲 **Использование библиотеки `random`**:
  - Отображение случайных фактов о погоде при запуске
  - Генерация случайных данных в тестах
